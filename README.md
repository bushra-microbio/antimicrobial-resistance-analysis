# Antimicrobial Resistance Analysis

## 👩‍🔬 Author
Bushra Mumtaz 

## 📌 Overview
This project presents a basic data analysis of antimicrobial resistance patterns in Klebsiella pneumoniae. It demonstrates the use of Python for handling biological datasets and visualizing resistance trends.

## 🎯 Objectives
- Analyze resistance rates of common antibiotics  
- Visualize antimicrobial resistance patterns  
- Build foundational skills in bioinformatics and data analysis  

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

## 📊 Dataset
A sample dataset representing resistance rates of selected antibiotics.

## 📈 Output
The script generates a bar chart showing resistance percentages across antibiotics.

## 🚀 Future Work
- Apply machine learning models (Random Forest, SVM)  
- Use real clinical or public datasets  
- Extend analysis to genomic or transcriptomic data  

## 📎 How to Run
1. Install Python  
2. Install required libraries:
   pip install pandas matplotlib  
3. Run the script:
   python amr_analysis.py
